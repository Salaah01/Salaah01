What are Primary and Foreign Keys?

A primary key is a column a database that uniquely identifies that record. For example, on a list of students details there may be column called studentID which their unique student number. The studentID would be the primary key as it is unique and can be used to identify any student.

A foreign key is a column a database table which is a primary key on another table. It is used to form relationships between multiple tables. For example, if you have a table with student test details, rather than using the students' names, you may have a studentID field which is defined in another table. This studentID column now enables you to be link an score back to a student.
